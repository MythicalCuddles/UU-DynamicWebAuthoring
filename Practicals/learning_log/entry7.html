Car Insurance Candidate Classification

The second task required creating a website for a Car Insurance Candidate Classification. The user would be required to input their information, and a calculation would be made to determine which class they belong to. As seen in Fig.3, all the inputs are stored in a variable. The age is then checked to be a valid (as in a number). If it is not, the user is alerted to this issue. The insurance class is then worked out by running if statements. The outside if checks the users age. Then nested if statements check to see if the user is married. Depending on which of these values are true will change the insurance class, which is saved to a variable and outputted at the end of the function.
