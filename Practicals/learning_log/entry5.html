Loops

There are many different ways to loop through data in JavaScript. We can use a for loop, do while loop and a while loop to loop through different variables such as arrays and numbers. Here are the functions I've created for this practical that allow the user to execute different kinds of loops (as seen in Fig. 1, 2 & 3)

User Display for Loops

For the user, they are able to click one of the three buttons at the top of the webpage (seen in Fig.4), which will execute their chosen loop and output the information to the console (seen in Fig.5).


We can also collect a users input and have a loop count to their input. By storing the value they input into the textbox in a variable, we can call the variable as part of the loop. The function seen in Fig.8 collects the value stored in the textbox to be used in the loop. Counting Backwards

We can also use loops to count backwards. Whilst the previously mentioned loops all increment a value to increase their number and checking condition, we are also able to decrease a number and checking condition. In Fig.9, I decided to add two textboxes for the user to input a high number and a lower number. The console will then output all the numbers to count down from the high number to the low number (as seen in Fig.10).
