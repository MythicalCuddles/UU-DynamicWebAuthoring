Working with Arrays

The second array is also created by instantiating the array class, but a length is not defined. This means that the array can have values stored in multiple different positions. Working with the Form

We can see in Fig.4 that we can create a form to allow users to input information into the input elements. Using the input elements, we can collect information such as the users name. We can also use select elements to allow the user to select from a list of options. In Fig.4, the options haven't been added in the HTML, but instead are added using JavaScript, which can be seen in Fig.5. JavaScript code can be run when the page has loaded by calling the onload event within the body element. In Fig.4, you can see that the onload event calls a function called onLoad(), which is seen in Fig.5.

In Fig.5, we can see that when the onLoad() function is called, the arrays are created. We can also loop through these arrays to create new elements on the webpage. This is what is being done to add the options for the dropdown boxes.

In Fig.6 and Fig.7, we have the Display button set to call the function displayInformation(). This function (in Fig.6) collects the information from the form, and displays text under the form using the information provided.
