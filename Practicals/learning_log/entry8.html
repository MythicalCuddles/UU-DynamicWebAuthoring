Inserting into onSubmit() & validAll() function (Parts II, VI, VIII & IX)

For this part, I amended the onSubmit() function to call a function called validAll(). All the functions created in this task are ran in there. If any of the functions return false, the code within validAll() will return false. This will then set allInfoValid = false, which will cause console.assert to run. If all the functions return true, none of the return false statements are ran within validAll(), and therefore will have allInfoValid set to true.
