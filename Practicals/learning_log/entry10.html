Image Rollovers

For the first task, I had to create a rollover image effect. I did this through using the onMouseOut event and onClick events of the image element, as seen in Fig.1. A live demo can be done here.

For the first task, making use of the onMouseOut event allowed me to execute a function everytime the users mouse left the boundaries of the image. This allowed me to run the function I created called userMouseOut(), which changes the image to the next image within the array, and keeps track of which image the user is seeing.

Due to the fact that the image that is being shown is stored as well, I can have an onClick event run, so when the user clicks on the image, a link can be retrieved from an array and the user can be directed to the unique link.


Slideshow

For this task, I made use of next and previous buttons, which are displayed on the sides of the image. A live demo can be done here.

When the user clicks the next or previous buttons, the nextImage() function is called. A parameter is passed to amend the value of onImage, which is the variable keeping track of what image is being shown to the user.

When the showImage() function is called through the next or previous buttons function, all of the images on the webpage are set to be hidden from sight by the use of a loop. The onImage variable then allows for the image that the user is to be viewing to be displayed.


Animated Advertising Banner

For this task, I used the setTimeout function to create a timer that calls the function changeImages every 3 seconds. The variable time can be set in seconds even though setTimeout accepts milliseconds as I am multiplying the value of time by 1000, which converts it to seconds.

Like task 1, shown in Fig 1 and 2, the images are stored in an array for this task. However, when the changeImages function is called, the image element on the webpage has the source attribute changed to a source link to the image. The variable keeping track of which image is being shown is also amended, before another timer is set to change the image again.
