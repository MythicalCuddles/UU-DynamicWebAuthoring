Design the quiz form

For the first part of the task, I designed a quiz using the design guidelines given in the task. Here, I created three select options for the user to select from a list of options to answer the questions.


Getting selected option value from form

Using the code seen in Fig.2, I can specify the name of the select I want to get the option value for, along with the name of the form. I have defaulted the form to use xhtmlQuiz as a default, therefore I do not need to mention it elsewhere in the code if I wish to use that form, which is the case for this task.


Checking the answers

Once the user has submitted their answers, the checkAnswers() function is ran (seen in Fig.3). Here, we first start with creating some arrays. Firstly, we need an array to store the users answers. I also created an array to store the correct answers, which has the values of the value attribute seen for the options in Fig.1. I also created a variable to store the score of the user.

Next, I run a room to loop through and get each of the questions selected answer value. These are all added to the users answers array. Doing this, I can check if any of the answers are unanswered by checking if the array includes an unanswered value. If the user hasn't answered a question, they are alerted of the issue and the function is exited.

I then selected to run a loop to compare the values of the answers array and the users selected answers. If their answer matches the correct answer, their score is incremented by one.

Lastly, I output the score to the user to let them know what they scored on the quiz.
