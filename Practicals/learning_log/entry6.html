Checking Grades

For this task, I created a form that allowed the user to input four values (Fig.8). The code will take these four values and calculate the average before outputting the result to the user. Once the user has entered their marks, they can click "Calculate Average" which will run a function to calculate their average mark and display it as an output, which can be seen in Fig.9.


The code seen in Fig.10 gathers the users inputs stores them into variables. A if statement allows for a check to be run, using a function isNaN(), which checks to see if the variable is not a number. If this is the case, an alert is sent, and a return statement prevents the function from continuing.

Another if statement is placed to see if the user has inputted numbers between 0 and 100. The user is able to input decimals into the textboxes, which will also work in the calculateAverage() function. If any of their inputted marks are above 100, or below 0, they are alerted that their input is invalid. And like above, a return statement is also used to stop the function from continuing.

After the checks have been preformed, the average mark is calculated by adding all the values together and dividing by the total amount of values given (in this case, 4). The average mark is outputted to the user.


I was able to use an if statement to cycle through multiple conditions to find the users grade. If the average mark falls within one of these branches, a grade is given and finally outputted to the user.


Like if statements, I can also use a switch statement to run through multiple conditions. As the grade is compared to each case, if the grade letter matches the case, the block within the case is ran. In Fig.12, a variable, feedbackText, is set to different values depending on which grade the user has from their average mark. This feedback is outputted to the user, as seen above.
