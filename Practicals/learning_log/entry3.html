Hands on with JS

For this task, the code seen in in Fig.2 was also given. The task involved adding JavaScript code inbetween the script elements. However, the use of the comments do not prevent the script from running. We can see two ways that comments are being used, which are '<!--', '-->' and '//', but these comment tags are limited in use. Usually, '<!--' & '-->'are used for HTML commenting, and are not used by JavaScript, which is why they show as errors. We are able to use '//' comments in JavaScript, which is the comment tag for JS, but only for single line comments.

We can see the output of the code from Fig.2 in Fig.3, which just writes "My age is 49" to the webpage.
