Introduction to JavaScript - ImageSwap

The webpage that I'm going to talk about in this section is available here.

The code seen in fig.4 shows an image and makes the image a link. The link uses two JavaScript events, which are 'onmouseover' and 'onmouseout'. These events are triggered using the mouse. Once the user moves their mouse over the image, the event 'onmouseover' is called, which finds the photo and modifies the source (src) to a new value. When the user moves their mouse out of the image, the event 'onmouseout' is called, finding the photo again and modifying the source again. The photo is able to be found in the document as the img tag has an attribute 'name', containing the value 'photo', which the events use. This action can be seen in fig.5 below.
